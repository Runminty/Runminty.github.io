What is the difference between Margin, Border, and Padding?
what is Margin

Margin is CSS properties are used to create space around elements, outside of any defined borders. There are properties for setting the margin for each side of an element (top, right, bottom, and left). --W3Shools

What is Border

The CSS border properties allow you to specify the style, width, and color of an element's border.--W3Shools

What is Padding

Padding is CSS properties are used to generate space around an element's content, inside of any defined borders. There are properties for setting the padding for each side of an element (top, right, bottom, and left).--W3Shools
